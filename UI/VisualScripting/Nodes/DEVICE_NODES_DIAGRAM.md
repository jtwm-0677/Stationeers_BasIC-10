## Color Coding Guide

When implementing UI:
- **Orange pins** (Device) - Thick border, warm color
- **Blue pins** (Number) - Cool color, most common
- **White pins** (Execution) - High contrast, always visible
- **Green pins** (Boolean True) - Success color
- **Red pins** (Boolean False) - Alert color
- **Purple pins** (String) - Uncommon, distinct
