## Issue 1: Code Pane Does Not Auto-Refresh on Property Edit

**Severity:** Medium

**Symptom:** When user edits node properties via UI TextBox, the Generated Code pane does not update to reflect changes. Code only regenerates when queried via MCP.

**Steps to Reproduce:**
1. Add NamedDevice node (shows default: `DEVICE device "StructureActiveVent"`)
2. Edit Alias Name to "sensor"
3. Observe Generated Code pane - still shows old value
4. Query code via MCP - now shows correct value

**Expected:** Code pane should auto-refresh when any property is edited
**Actual:** Code pane only updates when accessed via MCP

**Note:** This is a regression or incomplete fix - code auto-refresh was working for node additions and wire connections, but not for property edits.

---

## Issue 2: Cursor Position Off by One

**Severity:** Low (cosmetic)

**Symptom:** When clicking on a TextBox to edit, the cursor appears one character before the end of the existing text, instead of at the end.

**Example:** Clicking at end of "test" places cursor between "s" and "t"

**Expected:** Cursor should appear at end of text (or at click position)
**Actual:** Cursor appears one character early
